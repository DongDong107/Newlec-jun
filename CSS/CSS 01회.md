5강. 선택자(Selectors) - 중요

- html 태그 속성마다 style을 써주면 추후 수정 시에 힘들어진다. => 집중화가 필요!
- CSS를 통한 집중화.
- style 태그를 통해서 한번에 모든 h1이나 div 스타일을 바꿔줄 수 있지만 다르게 하고 싶을 때는 할 수 없어서 선택자가 필요하다.
